About This Project

Predictive Maintenance Dashboard demonstrates real-time monitoring, ML-driven failure prediction, and alerting for energy equipment.

  • Flask backend with REST APIs
  • RandomForest model for failure prediction
  • SQLite storage for historical trends
  • Bootstrap 5, Chart.js frontend
